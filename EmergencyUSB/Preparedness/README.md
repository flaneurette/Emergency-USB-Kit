# Prepardness files

Put your documents in this folder. Such as: government advice on preparedness, phone numbers, (surival kit) instructions, leaflets, emergency shelter locations.

### Ukraine advisories

Dedicated to Ukraine. These websites have some really good **war-related** preparedness documents, which can be helpful for other countries as well:

* https://dsns.gov.ua
* https://dovidka.info/

English:
  
* https://dsns.gov.ua/en
* https://dovidka.info/en/

The above sites have practical advise as what and **what not to do** in a war situation.
