# Offline AI

AI becomes indispensable nowadays. 

For offline AI access we can utilizes Ollama.

### Website / download: 

https://ollama.com/

What it does: Lets you download AI models and run them fully offline.

### Hardware notes:

Some models run on CPU only (slower but universal)

Others need a GPU for reasonable speed

### Use cases:

- Summarizing offline documents

- Code generation or debugging without internet

- Quick research or writing tasks in emergencies
